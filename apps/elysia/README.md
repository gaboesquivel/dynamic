# Elysia Template

Elysia template/example app for reference and experimentation.

## Overview

This is a minimal Elysia application template that demonstrates basic patterns and can be used as a starting point for new Elysia-based services.

## Tech Stack

- **Elysia** - Fast, functional web framework
- **Bun** - Runtime and package manager
- **Zod** - Schema validation
- **Vitest** - Testing framework

## Getting Started

### Prerequisites

- Bun >= 1.3.2
- Node.js >= 20

### Running Locally

```bash
# Development mode
bun run dev

# Build for production
bun run build

# Start production server
bun run start
```

## Development

```bash
# Lint
bun run lint

# Test
bun run test

# Test with coverage
bun run test:cov
```
